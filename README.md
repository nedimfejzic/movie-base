### Installation

1. Get a free API Key at [OMDb API](http://www.omdbapi.com/)
2. Clone the repo
   ```sh
   git clone https://github.com/nedimfejzic/movie-base.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Enter your API in `movieApiKey.js`
   ```js
   export const APIKey = "******";
   ```
